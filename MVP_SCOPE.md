# MVP Scope

Build the first version of the Life Journey Dashboard.

Use mock data only.
Do not build backend yet.
Do not add authentication yet.
Do not add database yet.

Focus on:
- Beautiful UI
- Clean architecture
- Useful dashboard pages
- Realistic mock data
- Responsive design
- Strong personal feeling

## MVP Pages

1. Dashboard
2. Today
3. Habits
4. Goals
5. Journal
6. Life Timeline
7. Weekly Review
8. Gym / Training

## Navigation

Sidebar items:
- Dashboard
- Today
- Habits
- Goals
- Journal
- Timeline
- Weekly Review
- Gym

Use Lucide icons:
- Dashboard: LayoutDashboard
- Today: CalendarCheck
- Habits: Repeat
- Goals: Target
- Journal: BookOpen
- Timeline: Milestone
- Weekly Review: ClipboardCheck
- Gym: Dumbbell

## Design

Follow design.md exactly.

Important:
- No gradients
- No extra accent colors
- Dark mode first
- Use #161614 as page background
- Use #1D1D1B for cards
- Use #E5E4DF for main text
- Use #7B7A74 for muted text and borders
- Use #5FB5D6 as the only accent
- Use JetBrains Mono for headings, labels, stats, nav
- Use Inter for body text
